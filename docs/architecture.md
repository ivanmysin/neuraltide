# Архитектура NeuralTide

## Обзор

NeuralTide — это библиотека для построения и симуляции нейронных сетей на основе популяционных моделей. Архитектура построена на следующих принципах:

1. **Модульность**: популяции, синапсы и генераторы — независимые компоненты
2. **Векторизация**: все операции используют TensorFlow для эффективных матричных вычислений
3. **Конфигурируемость**: параметры могут быть скалярами, векторами или обучаемыми переменными с ограничениями

## Workflow использования

### 1. Создание популяций

Популяции описывают динамику групп нейронов. Каждая популяция имеет:
- `n_units` — количество независимых единиц (векторных каналов)
- `dt` — шаг интегрирования в мс
- `params` — словарь параметров

```python
pop = IzhikevichMeanField(
    dt=0.5,
    n_units=4,
    params={
        'tau_pop': [1.0, 1.0, 1.0, 1.0],
        'alpha': [0.5, 0.5, 0.5, 0.5],
        # ... другие параметры
    }
)
```

### 2. Создание входных генераторов

Генераторы создают внешние входные сигналы. Они оборачиваются в InputPopulation и подключаются к динамическим популяциям через синапсы.

```python
gen = VonMisesGenerator(
    dt=0.5,
    params={
        'mean_rate': 20.0,
        'R': 0.5,
        'freq': 8.0,
        'phase': 0.0,
    }
)
```

### 3. Создание синапсов

Синапсы описывают проекции между популяциями:
- `n_pre` — число пресинаптических популяций
- `n_post` — число постсинаптических популяций
- `pconn` — матрица вероятностей соединений [n_pre, n_post]

```python
syn = TsodyksMarkramSynapse(
    n_pre=1,
    n_post=4,
    dt=0.5,
    params={
        'gsyn_max': 0.1,
        'tau_f': 20.0,
        'tau_d': 5.0,
        # ... другие параметры
    }
)
```

### 4. Построение графа сети

```python
graph = NetworkGraph(dt=0.5)
graph.add_input_population('theta', gen)      # входная популяция
graph.add_population('exc', pop)              # динамическая популяция
graph.add_synapse('theta->exc', syn, src='theta', tgt='exc')
```

### 5. Создание и запуск NetworkRNN

```python
network = NetworkRNN(graph, integrator=RK4Integrator())

# Симуляция
t_seq = tf.constant(np.arange(0, 1000, 0.5)[None, :, None])
output = network(t_seq)
```

## Параметры: система _make_param

Все модели используют единую систему регистрации параметров через `_make_param()`. Поддерживаются форматы:

### Скаляр (broadcast к n_units)
```python
params={'rate': 10.0}
```
Автоматически расширяется до `[10.0, 10.0, ...]`

### Вектор (по одному значению на единицу)
```python
params={'rate': [10.0, 15.0, 20.0]}
```

### Словарь с полным контролем
```python
params={
    'rate': {
        'value': 10.0,
        'trainable': True,
        'min': 0.0,
        'max': 100.0,
    }
}
```

## Семантика матрицы pconn

В синапсах матрица `pconn` имеет форму `[n_pre, n_post]`:
- `pconn[i, j]` — вероятность/сила связи **от** популяции `i` **к** популяции `j`

Пример: 2 входные группы → 2 нейрона
```python
pconn = [[1.0, 0.0],   # вход 0 → нейрон 0 (сильный), → нейрон 1 (слабый)
         [0.0, 1.0]]   # вход 1 → нейрон 0 (слабый), → нейрон 1 (сильный)
```

## Типы данных

NeuralTide использует TensorFlow для всех вычислений. Глобальный dtype можно настроить:

```python
import neuraltide.config as config
config.set_dtype(tf.float64)  # по умолчанию tf.float32
```

## Симуляция через tf.scan

NetworkRNN использует `tf.scan` для эффективной симуляции на временной оси. На каждом шаге:
1. Вычисляются синаптические токи от всех проекций
2. Обновляются состояния популяций через интегратор
3. Обновляются состояния синапсов

Это обеспечивает:
- Векторизацию всех операций
- GPU-ускорение
- автоматическое дифференцирование для обучения
